# Skinature

Premium natural **skincare & haircare** e-commerce — a complete rebuild of
[skinature.org](http://skinature.org) on Next.js, launching at **skinature.com**.

> _Nurtured by Nature._

## Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion · Lenis
· Supabase · Razorpay · Resend + react-pdf.

## Getting Started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm run start    # start production server
npm run lint     # ESLint
```
